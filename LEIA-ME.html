O projeto foi desenvolvido utilizando a linguagem Java SE:

O arquivo : Calculadora_Aritmetica_Binaria.jar é o executável do projeto.

O arquivo: Calculadora_Aritmetica_Binaria.rar é a pasta do projeto

OBS¹: O projeto foi desenvolvido utilizando a IDE NetBeans 8.1

OBS²: É recomendado que caso deseje verifica código código fonte do projeto, utilize a IDE NetBenas

Pacotes e arquivos do projeto

Pacote: view contem o seguinte arquivo:

  • ViewCalculadora.java - Responsável para carrega a interface do projeto

Pacote: controller contem os seguintes arquivos:

  • Conversor.java - Responsável para fazer as conversões do projeto como, por exemplo, de hexadecimal para binário, decimal para binário
  • Operacao.java - Responsável para fazer as operações aritmetica do projeto, como por exemplo, adição, subtração, multiplicação
